# avaughan89.github.io

Personal one-page website for Andrew Vaughan, built as a lightweight static site for GitHub Pages.

## Structure

- `index.html` - semantic page content and metadata
- `index.css` - responsive styling and layout
- `imgs/headshot2026.jpg` - headshot used by the hero profile panel and social preview

No build step or framework is required. GitHub Pages can serve the site directly from the repository root.
